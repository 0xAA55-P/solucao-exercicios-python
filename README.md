## Estrutura do Repositório

```
Simples/                     # Exercícios Simples
    Datas/       # Com Manipulação de Datas
    Funcoes/     # Com Funções
    Dicionarios/ # Com Dicionários
    Listas/      # Com Listas
    Mensagens/   # Input e Output
    Numeros/     # Operações Aritméticas
    Strings/     # Operações Com Strings

Avancado/                   # Exercícios Avançados
    Lambda/      # Funções Lambdas
    Map/         # Com Maps
    Reduce/      # Com Reduce

```

## Como executar

- Você precisa ter o Python 3 instalado.

```bash
# Clone o repositório
git clone https://github.com/0xAA55-P/solucao-exercicios-python

# acesse uma pasta e execute um exercício
cd Simples/Listas
python 001.py
```
